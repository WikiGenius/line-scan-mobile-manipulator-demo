# Source

This folder holds public-safe source files for simplified toy examples.

Current file:

```text
toy_line_scan_coverage_demo.py
```

The toy demo accumulates scan-line coverage over a synthetic grid and writes a CSV table plus an SVG figure. It is not a mobile-manipulator planner and does not model real line-scan or hyperspectral sensor physics.

Unpublished planning algorithms, private data, and paper-specific implementations should remain private until release is approved.
